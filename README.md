Sentiment Analysis & Emotion Detection App:
Project Description:
This project is a Sentiment Analysis system that analyzes text data to determine:
  Sentiment: Positive, Negative, or Neutral
  Emotions: Joy, Anger, Sadness, Fear, Surprise, Neutral
The system uses Groq LLM, NLP techniques, and lexicon-based analysis to process data from multiple sources such as Amazon reviews, social media text, and news articles.
It helps understand public opinion and sentiment trends and supports marketing, product, and social insights.

Technologies Used:
Python
Streamlit – Web interface
Groq LLM (LLaMA 3) – Deep sentiment & emotion analysis
TextBlob – Lexicon-based sentiment analysis
dotenv – Environment variable management
Pandas – Data processing
Matplotlib – Visualization
